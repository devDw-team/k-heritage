인터랙션 & 애니메이션

터치 피드백

  • 카드 호버: 상승 효과 + 그림자
  • 버튼 클릭: 스케일 축소 후 복원
  • 칩 선택: 색상 전환 애니메이션
  • 리스트 스크롤: 관성 스크롤

화면 전환

  • 페이지 이동: 슬라이드 인/아웃
  • 모달: 페이드 인 + 스케일
  • 하단 시트: 드래그 제스처
  • 이미지 로딩: 스켈레톤 → 페이드 인

마이크로 인터랙션

  • FAB 펄스: 주목도 향상
  • 마커 바운스: 선택 시 반응
  • 스위치 토글: 부드러운 전환
  • 로딩 스피너: 한국 전통 문양

접근성 고려사항

시각적 접근성

  • WCAG AA 기준 색상 대비 (4.5:1 이상)
  • 최소 터치 영역 48x48dp
  • 폰트 크기 조절 지원
  • 다크 모드 완벽 지원
  • 아이콘과 텍스트 레이블 병행

스크린 리더 지원

  • 의미 있는 대체 텍스트
  • 논리적인 포커스 순서
  • 역할 기반 시맨틱 마크업
  • 상태 변경 음성 안내
  • 제스처 대체 방법 제공
